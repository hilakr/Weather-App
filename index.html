Weather-app

Weather App allows other developers have an easy and orderly way to get weather data info around the world using two different web service.

The Weather App is avilable libary written in c# which is a wrapper libary for working with restful web services, I choose 2 web services that supply weather data info : 1.http://www.openweathermap.org/api 2.http://www.worldweatheronline.com/api/

If you want to perform weather Analysis, but don't wish to do the data scraping yourself, you can consider using weatherData. Given a few parameters, it has functions that return the available data in a time-stamped data frame that is easy to work with.

The main goal of the libary is to help developers with working with restful web services,if you want to perfom weather analysis, but don't wish to work directly with the data , you can use the weather-app to do it for you :)
